# Card Memory Game
My first web project. A simple browser-based card matching game built with JavaScript.

**Card Assets:** *[Pixel Art Cards](https://opengameart.org/content/pixel-art-cards)* by Lempf (Glenn Dittmann)

**Sound Effects:**
* Card sounds made using [sfxr.me](https://sfxr.me/)
* applause.wav original: [Well Done](https://opengameart.org/content/well-done) by qubodup

**Music:**
* *Classic Theme* - Cover (made by me) of the Forest Theme from *Power Rangers: Samurai* (Wii/DS Game). Appropriate credits for the original can be found in the description of [this YouTube video](https://www.youtube.com/watch?v=n2MxxBpqUB0)

**Fonts:**
* *[Born2bSporty](https://www.1001fonts.com/born2bsporty-fs-font.html)* by UkiyoMoji
* *[Hydrophilia Iced Regular](https://www.1001fonts.com/hydrophilia-font.html)* by Floodfonts
